# Docs

This folder captures the Extensity 2.0 rollout from three angles:

- `extensity-2.0-plan.md`: the implementation plan and the architectural decisions behind it.
- `extensity-2.0-status.md`: what was actually implemented in the current branch, including gaps and deferred work.
- `ci-and-release.md`: how CI, build packaging, and Chrome Web Store submission artifacts work.

For the high-level product overview, see the repository root `README.md`.
For local build steps, see `BUILD.md`.
